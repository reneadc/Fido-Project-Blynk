Fido  Senior Project - Made in Summer 2019  - Contributor: Reneanel De Castro


Fido is an IOT project using a Raspberry Pi. Fido is an automatic dog feeder controlled with the Blynk App

Python for the Raspberry PI

Items needed for project:

1)Raspberry Pi (Model B) or higher
2)Breadboard for circuit connections
3)Pi Camera
4)Servo Motor 5v and 180 rotation
5)LCD 12C 16 bit or 20 bit display
6) GPIO Breackout board




Executable files:

motor.py to run servo motor and LCD output display
